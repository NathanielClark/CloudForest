  • CloudForest

    Version 0.1 beta

    CloudForest generates phylogenetic trees on the cloud using MapReduce. CloudForest is designed to be run on Amazon's Elastic MapReduce servers. However, it is possible to run it on your own cluster or even on your local machine.
